/*웹 애플리케이션 개발 순서*/


1. 데이터 베이스 처리
  ㄴ 스키마 생성 -> 테이블 -> 기능 구현을 위한 sql 생성

2. 프로젝트 생성 : JRE, Runtime(톰켓)

3. 라이브러리 추가 : /WEB-INF/lib에 추가
  ㄴ 데이터베이스 드라이버, JSTL, 파일 업로드, JSON, XML 등등

4. 애플리케이션 초기화
  ㄴ 서블릿 => 데이터베이스 드라이버 로드

5. 기능 클래스 정의
  ㄴ Connection, Close

6. 사용자 요청 기능 구현
  - JSP(요청을 받는 기능)
  - Service Class(기능처리)
  - Domain(데이터 저장용)
  - DAO(데이터 베이스 기능 처리)
  - JSP(응답을 표현 View)
